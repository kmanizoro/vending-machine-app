# Vending Machine App
Simple vending machine application

#### Features!
  - Choose a products like Chocolate, Candy, Cold Drink and Sweet Bread
  - Insert a coins like Cent-1, Nickle-5, Dime-10, Quater-25;
  - Get the choosen product with balances
